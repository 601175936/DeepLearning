project: Gesture recognition using CNN and radar signal processing

	This project proposed a method to recognize gesture by using convolution neural network 
	based on radar signal processing, test accuracy is 100%.
	
file:
	convert_rgb_resize.py : convert original images to RGB style and resize it. 
	generate_dataset.py   : generate dataset using RGB images which have been resize
	main.py               : train a modle to classify images
